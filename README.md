Desafio Mks
===========================

## Instruções

Olá, esta é a minha entrega oficial para o desafio da MKS
espero que goste :)

1. Depois de clonar o projeto, basta executar:
```
docker compose up --build
```
2. Para acessar a documentação acesse: http://localhost:8080/docs
